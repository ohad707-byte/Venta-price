# VENTA - ממשק אנשי מכירות + דשבורד מנהל

## מה יש בפרויקט
- `public/index.html` - ממשק אנשי מכירות בלבד. אין דשבורד ואין עריכה.
- `public/admin.html` - דשבורד מנהל לעדכון מוצרים ומבצעים.
- `data/products.json` - בסיס הנתונים המרכזי של המוצרים והמבצעים.
- `server.js` - שרת קטן שמאפשר קריאה לכולם וכתיבה רק למנהל.

## הפעלה במחשב לבדיקה
```bash
npm install
ADMIN_PASSWORD=123456 npm start
```

פתיחה:
- אנשי מכירות: `http://localhost:3000`
- מנהל: `http://localhost:3000/admin.html`

## העלאה לשרת
הכי קל: Render / Railway / VPS.

הגדרת חובה בשרת:
```bash
ADMIN_PASSWORD=הסיסמה_של_המנהל
```

## איך זה עובד
- אנשי המכירות קוראים אוטומטית את `products.json` מהשרת.
- המנהל נכנס ל־`admin.html`, מעדכן מבצעים ושומר.
- השמירה נכתבת לשרת.
- כל איש מכירות שפותח/מרענן את הממשק מקבל את הנתונים המעודכנים.

## חשוב
לא לשלוח לאנשי מכירות את הקישור `admin.html` ולא את סיסמת המנהל.
